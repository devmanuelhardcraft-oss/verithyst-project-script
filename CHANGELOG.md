# Verthyst-Script Patch Notes

## Version 1.0 - Fedora Gaming Edition

### Added
- Added Fedora compatibility improvements
- Added better user permission handling
- Added user-based configuration storage
- Added MangoHUD user configuration support
- Added GameMode and gaming profile improvements
- Added language selection system
- Added developer language reset support

### Changed
- Improved gaming scripts for Fedora systems
- Improved optimization script compatibility
- Improved tuned and power profile handling
- Changed language settings to save per user
- Improved Steam/Discord permission compatibility

### Fixed
- Fixed root-owned user configuration files
- Fixed Steam permission issues after installation
- Fixed MangoHUD configuration ownership
- Fixed Fedora package conflicts with power profiles
- Fixed installer script structure

### Notes
This release focuses on making Verthyst-Script more reliable on Fedora-based gaming systems.
